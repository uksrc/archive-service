IVOA TAP Query Examples

Example 1: Simple Select Query

This example demonstrates a basic TAP query to retrieve all tables.

        SELECT TOP 10 * FROM TAP_SCHEMA.tables

This query selects all the tables from the TAP_SCHEMA.tables.

Example 2: Simple Select Query 2

This example demonstrates a basic TAP query to retrieve all tables from a specific namespace.

        SELECT * FROM TAP_SCHEMA.tables
        WHERE schema_name='public';

This query selects all the tables from the selected schema_name.

Example 3: Simple Select Query 3

This example demonstrates a basic TAP query to retrieve columns from a specific table.

        SELECT * FROM TAP_SCHEMA.columns
        WHERE table_name = 'observation';

This query selects all the tables from the TAP_SCHEMA.tables.

Example 4: Limited Query

This example shows a query with that limits the results to the first 5 entries.

        SELECT TOP 5 * FROM Observation;

This query retrieves all properties from the top 5 entries of Observation.

Example 5: Constrained Query

This example shows a query that constrains the results to those that are linked to a specific telescope.

        SELECT o.* FROM Observation o
        JOIN Telescope t ON o.telescope_id = t.id
        WHERE name = 'Lovell';

This query retrieves all entries of Observation where the Telescope used is the name supplied.

Example 6: Bounded Query

This example shows a query that constrains the results to a boundary.

        SELECT o.* FROM Observation o
        JOIN Telescope t ON o.telescope_id = t.id
        WHERE geoLocationX > 49.9 AND geoLocationY > 60.9
        AND geoLocationX < -8.1 AND geoLocationY < 1.8;

This query retrieves all entries of Observation where the Telescope used is located within a boundary.
